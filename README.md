# Amazon Sales Data Analysis & Dashboard – 2025 | SQL + Power BI

## Project Overview
End-to-end analysis of Amazon sales data (15,000+ orders, ₹1.12B revenue) using SQL and Power BI. Focuses on sales trends, customer behavior, returns, and regional insights.

## Key Features
- Data cleaning & validation using SQL
- Sales & category analysis to identify top-performing products and high-return items
- Customer segmentation to highlight top 25% high-value customers
- Returns & trend analysis with seasonality and regional insights
- Interactive Power BI dashboard with KPIs, monthly trends, returns tracking, and state-level performance

## Skills & Tools
SQL, Power BI, Data Analysis, Dashboard Design, Customer Segmentation, KPI Tracking

## Impact
Provides actionable insights to reduce returns, improve customer satisfaction, and optimize revenue.
